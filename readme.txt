WP Installer helps new WP installations by configuring the initial setup through 5 simple steps without setting each section

== Description ==
WP Installer is intended for new WP installations.
Instead of browsing each section to configure your new blog, you can easily do it through simple steps:

1-  **Manage Default Posts/Pages**

* Delete "Hello Worlds!" post

* Delete "Hello Worlds!" comment

* Delete "Sample Page!" page  

* Delete all posts

* Delete all pages

* Delete all comments

* Delete all categories & tags

* Delete all medias 

2- **Manage Default Widgets**

* Delete Recent Posts widget

* Delete Recent Comments widget

* Delete Archives widget

* Delete Categories widget

* Delete Search widget

* Delete Meta widget

* Delete all sidebar widgets 

3- **Theme Management & Activation**

* Upload a Theme File (zip)

* Upload a Theme File (from URL)

* Activate a Defined Theme

4- **Plugin  Management & Activation**

* Upload a Plugin  File (zip)

* Upload a Plugin File (from URL)

* Activate a Defined Plugin

5- **Add Custom Pages**, plus predefined

* About Us

* Contact Us

* Privacy Policy

* Terms of Use

* Earning Disclaimer 

6- **Set General Settings**

* Set Title

* Set Description

* Set permalinks

* Set Notifications

* Set Front Page Display

* Set Post List


== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/WordpressInstaller  directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
